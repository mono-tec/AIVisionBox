# AIVisionBox.Core

AIVisionBox における **中核（Core / 契約層）** を定義するプロジェクトです。

## 役割

- AI推論・画像処理・通信方式に依存しない **共通インターフェース**
- アプリケーション全体で使用される **データモデル（契約）** の定義
- 長期運用を前提とした **安定性・保守性重視の層**

本プロジェクトは **最も重要な基盤** であり、実装依存を一切持ちません。

---

## 含まれる主な要素

- `IAiInferenceEngine`  
  推論エンジンの共通インターフェース

- `ImageData`  
  推論エンジンへ渡す画像データの共通表現

- `InferenceResult`  
  推論結果の共通表現（PLC連携・HMI表示の基準）

- `Detection`  
  物体検出（Object Detection）結果の1件分モデル

- `ImageFormat`  
  画像フォーマット列挙型

---

## 設計方針

- `.NET Standard 2.1`
- C# 8 準拠（`init` / `record` は使用しない）
- コンストラクタによる初期化を基本とし、不変性を重視
- 外部ライブラリ（OpenCV / HTTP / SDK）への依存は禁止

---

## 注意

- 本プロジェクトには **実装コード（画像処理・通信処理）を追加しない**
- 機能追加は、必ず Engine 層または App 層で行う
